# Student Project Showcase — Design Brainstorm

## Response A — Soft Stationery
<response>
<text>
**Design Movement:** Soft Stationery / Washi-tape Scrapbook
**Core Principles:**
- Warm cream/off-white base with hand-drawn-style accents
- Bubbles feel like paper cutouts with gentle drop shadows
- Typography mixes a rounded display font with a clean sans-serif body
- Interactions feel tactile — slight "press" scale on click

**Color Philosophy:** Each student bubble draws from a curated set of 20 warm pastels (peach, sage, lavender, butter, blush, mint, sky, lilac, coral, rose…). Colors feel like highlighter markers on cream paper — cheerful but never garish.

**Layout Paradigm:** Scattered organic arrangement on the home page — bubbles float in a loose mosaic, not a rigid grid. Gallery page uses a clean 3-column photo grid with rounded corners and soft borders.

**Signature Elements:**
- Subtle paper-grain texture on the background
- Thin hand-drawn border rings around each bubble
- Dotted or dashed dividers between sections

**Interaction Philosophy:** Gentle — hover lifts bubble with a soft shadow bloom; click causes a satisfying "pop" scale animation before navigating.

**Animation:** Bubbles float in with a staggered fade+scale on page load. Gallery thumbnails slide in from below. Lightbox fades in with a slight zoom.

**Typography System:** `Nunito` (rounded, friendly) for names and headings; `DM Sans` for body text and buttons.
</text>

## Response B — Bauhaus Playful
<response>
<text>
**Design Movement:** Neo-Bauhaus / Playful Geometry
**Core Principles:**
- Bold geometric shapes with flat pastel fills
- Strong typographic hierarchy — names are large and confident
- Asymmetric layout with intentional negative space
- Color blocks define zones rather than decorative borders

**Color Philosophy:** 20 distinct flat pastels drawn from a Bauhaus-inspired palette — each is a pure, slightly desaturated hue. Background is a very light warm grey (#F7F5F2). Teacher mode uses a deep charcoal accent.

**Layout Paradigm:** Home page uses a masonry-style staggered grid of circles at varying sizes (all same size but offset rows). Gallery page has a strict 3-column grid with bold color header bar matching the student's bubble.

**Signature Elements:**
- Thick colored border ring on each bubble that matches the fill
- Bold uppercase student name inside the bubble
- Full-width colored header stripe on gallery page

**Interaction Philosophy:** Confident — hover rotates bubble slightly and deepens shadow; click triggers a bold color flash before navigation.

**Animation:** Page load: bubbles cascade in from top with a bounce. Gallery: grid items pop in with a stagger. Lightbox: slides in from the right.

**Typography System:** `Poppins` (geometric, bold) for names/headings; `Inter` for UI labels and buttons.
</text>

## Response C — Dreamy Watercolor
<response>
<text>
**Design Movement:** Dreamy Watercolor / Soft Studio
**Core Principles:**
- Very light, airy feel — almost white background with soft color washes
- Bubbles feel like watercolor blobs — slightly irregular, soft-edged
- Typography is elegant and light — thin weights for names
- Everything breathes — generous padding and whitespace

**Color Philosophy:** 20 soft watercolor pastels — each bubble has a radial gradient from a light tint to a slightly deeper version of the same hue. Background is pure white. The overall impression is a color-washed art studio wall.

**Layout Paradigm:** Home page: centered flowing layout with bubbles arranged in 4 rows of 5, evenly spaced. Gallery page: 3-column photo grid with a soft colored wash header. Clean and calm.

**Signature Elements:**
- Radial gradient fill on each bubble (light center → deeper edge)
- Soft blurred drop shadow on bubbles
- Subtle watercolor splash SVG behind the bubble grid

**Interaction Philosophy:** Dreamy — hover causes bubble to gently pulse/breathe (scale 1.05 with ease-in-out); click fades to gallery with a soft dissolve.

**Animation:** Bubbles drift in with a gentle fade+float-up stagger. Gallery thumbnails appear with a soft fade. Lightbox opens with a smooth zoom-fade.

**Typography System:** `Quicksand` (rounded, airy) for student names; `DM Sans` for UI elements.
</text>

</response>
